# `wasm-pack-action`

Install `wasm-pack` by downloading the executable (much faster than `cargo install wasm-pack`, seconds vs minutes).

## Patch

| before | now |
| :-: | :-:  |
| node16 | node20 |
| eslint | oxlint |
| @zeit/ncc | @vercel/ncc |

## Usage

```yaml
- uses: qmaru/wasm-pack-action@v0.5.2
  with:
    # Optional version of wasm-pack to install(eg. 'v0.13.1', 'latest')
    version: 'latest'
```
